# Карта маршрутов и якорей

## Якоря главной страницы
- `#services` → секция «Услуги»
- `#about` → секция «О компании»
- `#geography` → секция «География»
- `#fleet` → секция «Наш автопарк»
- `#contacts` → секция «Связаться с нами»

## Основные страницы
- `/` → главная
- `/calculator` → расширенный калькулятор грузоперевозки
- `/request` → расширенная форма запроса / отправки коммерческого предложения
- `/contacts` → отдельная страница контактов
- `/faq` → FAQ

## Страницы услуг
- `/services/intercity` → междугородние перевозки
- `/services/interterminal` → межтерминальные перевозки
- `/services/expedition` → экспедиционное направление
- `/services/project` → проектные перевозки
- `/services/adr` → опасные грузы

## Юридические страницы
- `/offer`
- `/privacy`
- `/terms`
- `/data-processing`

## Внешние маршруты
- `https://ati.su/` → профиль ATI
